## Smallfuck

```
Smallfuck是在2002年3月由Nikita Ayzikovsky发明的。
Smallfuck可以视作是Brainfuck的一个简化版语言，与Brainfuck不同，Smallfuck的操作只能对0/1进行反转。
```

#### Try It

```
$ python smallfuck.py '>*>*' '00101100'
```

#### Run Tests
```
python tests.py
```
