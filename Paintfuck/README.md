## Paintfuck 

```
这也是Brainfuck的一个变种，输出结果是以矩阵图的形式呈现的。
```

#### Try It

```
$ python paintfuck.py '*e*e*e*es*es*ws*ws*w*w*w*n*n*n*ssss*s*s*s*' 100 6 9 
```

#### Run Tests
```
python tests.py
```
